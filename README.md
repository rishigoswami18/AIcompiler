# 💼 CRM System

A full-featured Customer Relationship Management system built with Streamlit, featuring login authentication, contacts management, company tracking, deal pipeline, activity logging, and an analytics dashboard.

## Features

- 🔐 **Authentication** – Secure login & registration with bcrypt password hashing
- 📊 **Dashboard** – Real-time KPIs, deal pipeline funnel, and revenue breakdown charts
- 👥 **Contacts** – Full CRUD with search & status filtering
- 🏢 **Companies** – Company management with industry tracking
- 💰 **Deals** – Kanban pipeline view + table view with stage filtering
- 📅 **Activities** – Task/call/email/meeting tracking with completion status
- ⚙️ **Settings** – Profile management & admin user management panel
